控件:控件框

控件:控件框



这些控件的运行类似于带标题版本的控件.框体控件里面包含了另一个类的控件.

BoxBasic
 BoxBasic会打印一个带有可选名称和页脚的框到屏幕上.它是用来作为其他控件的基础类的,而不是拿来直接使用的.
BoxTitle
 BoxTitle是一个标题控件和多行控件的混合体.同样的,它也是很大程度上准备用来作为 布局更加复杂的其他类 的基础类.这个类有一个 _contained_widget 属性,会在该类被创建时往实例中放进一个控件到里面.在 BoxTitle类中,它是一个 MultiLine控件.控件的标题可以用参数 name=... 传递到 __init__ 里,另一个参数 footer=... 给出了框体文本的页脚.这里与 name 和 footer 名字对应的属性可能会在任何时候被修改.
 entry_widget 属性提供了直接对 内部所包含的控件 的访问.
 属性 editable, values, 和 value 给出了直接对 entry_widget 属性的访问.
 该控件的构造函数可以接受 contained_widget_arguments 参数.它需要是一个字典参数,在创建时它将会被传递到 entry_widget.注意,这回对这个字典是没有完整性检查的.(在4.8.0版本中新加入)

你自己版本的这些控件可以用与新的标题控件的相同方式被创建.首先创建里面包含的控件,然后再创建框体类封装的类:
class NewMultiLineClass
        # Do all sorts of clever things here!
        # ....

 class BoxTitle(BoxBasic):
     _contained_widget = NewMultiLineClass
