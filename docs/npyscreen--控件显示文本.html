控件:显示文本



Textfild,TitleText
 一个单行的文本,虽然长度是任意的 -它一个基本的输入控件.注意,有一些版本的文档中会不恰当的将其指代成 'textbox'(文本框)控件.
FixedText, TitleFixedText
 一个单行文本,但其文本框的编辑功能被移除了.
Autocomplete
 这是一个带有附加功能的文本框-其概念是如果用户按下了 TAB键控件会尝试'补全'用户正在输入的内容,适当的给出可供选择的选项.其调用的方法是 auto_complete(inputch).
 当然,上下文就是一切. Autocomplete也因此没那么有用,但其目的是作为你能用作子类的东西.参见 Filename 和 TitleFilename 这两个类作为例子.
Filename,TitleFilename
 一个会尝试'补全'用户输入的文件名或者路径的文本框.
 这是一个 Autocomplete 控件使用的例子.
MultiLineEdit
 这是一个允许用户编辑多行文本的控件.
Pager,TitlePager
 该控件显示多行文本,并允许用户来回滚动,但不能编辑.要显示的文本保存在 .value 属性中.[页面控件]

详解



class Textbox
 display_value(vl)
  控制 .value 属性的值如何显示.因为多个版本的文本控件相互使用,混合控件(比如大多数的多行控件类)里,该方法通常会被覆盖重写.
 show_brief_message()
  发出蜂鸣并显示一个短信息给用户.通常是有更好的方式来做这些的,但是它也是有用的,比如在 Autocomplete 类中显示错误.
